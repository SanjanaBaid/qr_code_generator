QR CODE GENERATOR

A simple and modern QR Code Generator Web Application built using HTML, CSS, and JavaScript.
This application allows users to enter text or a URL and instantly generate a QR code.

FEATURES

* Generate QR codes from text or URLs**
* Responsive layout
* Input validation
* Toast notifications for user feedback
* Smooth button interactions
* QR code preview directly on the page

TECHNOLOGIES USED

* HTML5 – Structure of the web page
* CSS3 – Styling and modern UI effects
* JavaScript – Functionality and QR code generation
* jQuery – Simplified DOM manipulation
* jQuery QRCode Library – Generates QR codes dynamically

HOW IT WORKS

1. The user enters text or a URL into the input field.
2. When the Generate QR Code button is clicked:
   * JavaScript captures the input value.
   * The QR code container is cleared.
3. The jQuery QRCode plugin generates a QR code using the input text.
4. The QR code is displayed inside the QR container on the page.

Created as a simple web project using HTML, CSS, and JavaScript.
